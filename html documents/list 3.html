HTML
Hypertext Markup Language is the standard markup language for creating web pages and web applications. With Cascading Style Sheets and Javascripts, it forms a triad of cornerstone technologies for the World Wide Web.
PHP
PHP is a server-side scripting language designed for web developement but also used as a general -purpose programming language. Originally created by Rasus Lerdorf in 1994, the PHP reference implementation is now produced by the PHP Group.
